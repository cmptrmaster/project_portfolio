Message Generator

The purpose of this little project was to pull a random message and present it to the user. A text file is used as a source of the messages, one message per line. The application imports the text file, and then randomly selects one to present to the user. The application was written in JavaScript and intended to be run using node.js. The text file can be modified to add or remove any of the messages.
